How to use


수능 학습을 위한 모바일 기반
PDF 간편 채점 학습 도우미 앱입니다.

1. 풀이를 원하는 PDF 문제파일을 업로드
2. 한 문제씩 문제를 풀 수 있는 화면이 제공되고 풀이와 답을 PDF에 직접 수기로 작성
3. 글씨를 인식해 채점 결과 제공
4. 문제풀이 후 점수와 연관 자료들을 확인
